## 💻 Development

### 🧰 Prerequisites

- yarn (v1.22.10)
- node (v16.2.0)

### ⚙️ Setting Up

- Install dependencies
  ```
  yarn install
  ```
- Run in development mode
  ```
  yarn start
  ```
